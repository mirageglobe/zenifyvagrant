Zenifywordpress dev env is a vagrant based development enviroment for wordpress. 

TOUSE
---------------------

To start:

> $bash vagrant.run.sh

If you update the vmbootstrap.sh file, you can refresh the vagrantvm by:

> $bash vagrant.refresh.sh
